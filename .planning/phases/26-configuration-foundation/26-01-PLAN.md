- Import: os, tempfile at top of file
- Create temp file in same directory as target config using tempfile.NamedTemporaryFile:
  - mode='wb'
  - dir=config_path.parent (CRITICAL: same filesystem for atomic replace)

- Atomically replace target using os.replace(temp.name, config_path)
- On exception: cleanup temp file with os.unlink(), re-raise
