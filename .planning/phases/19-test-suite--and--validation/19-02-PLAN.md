**Specific tests to write:**

1. `test_window_cleanup_after_manual_close`:
   - Create terminal for case X
   - Verify window ID registered
   - Manually close window (via AppleScript close command)
   - Run attach_terminal again for same case
   - Verify: New window created (not error), old entry cleaned up

2. `test_stale_window_id_handling`:
   - Register fake window ID in WindowRegistry
   - Verify window doesn't actually exist (validation fails)
   - Run attach_terminal for that case
   - Verify: Stale entry removed, new window created

3. `test_registry_corruption_graceful_fallback`:
   - Create container and window
   - Corrupt/delete registry database file
   - Run attach_terminal again
   - Verify: New registry created, window created (no crash)
