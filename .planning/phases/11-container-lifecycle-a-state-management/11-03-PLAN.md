**Method signature:**
```python
def list(self) -> list[dict[str, Any]]:
    """List all mc-managed containers with status and metadata.

    Returns:
        List of container info dictionaries with keys:
        - case_number: str
        - status: str (running, stopped, exited, etc.)
        - customer: str (from label or "Unknown")
        - container_id: str (short ID)
        - workspace_path: str (from state database)
        - created_at: str (formatted timestamp)
        - uptime: str (if running, e.g., "2h 34m")

    Raises:
        RuntimeError: If Podman connection fails
    """
```

**Implementation:**
1. Call self._reconcile() to sync state with Podman
2. Query Podman for all mc-managed containers:
   ```python
   containers = self.podman.client.containers.list(
       all=True,  # Include stopped
       filters={"label": "mc.managed=true"}
   )
   ```
3. For each container:
   - Extract case_number from container.labels["mc.case_number"]
   - Get status from container.status
   - Get customer from container.labels.get("mc.customer", "Unknown")
   - Get short_id from container.short_id
   - Query state database for metadata: self.state.get_container(case_number)
   - Extract workspace_path from metadata (or "N/A" if not in state)
   - Format created_at timestamp: use metadata.created_at if available, else "Unknown"
   - Calculate uptime if status == "running": use container.attrs["State"]["StartedAt"] to compute duration
4. Sort by created_at timestamp (newest first)
5. Return list of dictionaries

**Uptime calculation helper:**
```python
def _calculate_uptime(self, started_at: str) -> str:
    """Calculate human-readable uptime from ISO timestamp.

    Args:
        started_at: ISO 8601 timestamp (e.g., "2026-01-26T12:34:56.789Z")

    Returns:
        Human-readable duration (e.g., "2h 34m", "5d 3h", "45s")
    """
    from datetime import datetime, timezone
    started = datetime.fromisoformat(started_at.replace('Z', '+00:00'))
    now = datetime.now(timezone.utc)
    delta = now - started

    days = delta.days
    hours, remainder = divmod(delta.seconds, 3600)
    minutes, seconds = divmod(remainder, 60)

    if days > 0:
        return f"{days}d {hours}h"
    elif hours > 0:
        return f"{hours}h {minutes}m"
    elif minutes > 0:
        return f"{minutes}m {seconds}s"
    else:
        return f"{seconds}s"
```

Handle edge cases:

- Container in state but not in Podman: reconcile() will remove it
- Container in Podman but not in state: include with workspace_path="N/A" (shouldn't happen if create() is used)
