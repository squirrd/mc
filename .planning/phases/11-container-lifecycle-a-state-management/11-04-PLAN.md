**stop() method:**
```python
def stop(self, case_number: str, timeout: int = 10) -> bool:
    """Stop running container gracefully.

    Args:
        case_number: Case number
        timeout: Seconds to wait for graceful shutdown before SIGKILL (default: 10)

    Returns:
        True if container stopped, False if already stopped

    Raises:
        RuntimeError: If container not found or stop fails
    """
```

**Implementation:**
1. Get container metadata from state: self.state.get_container(case_number)
2. If not in state, raise RuntimeError(f"No container found for case {case_number}")
3. Get container from Podman: container = self.podman.client.containers.get(metadata.container_id)
4. Check status: if container.status in ("stopped", "exited"), return False
5. Call container.stop(timeout=timeout) (sends SIGTERM, waits, then SIGKILL)
6. Return True

**delete() method:**
```python
def delete(self, case_number: str, remove_workspace: bool = False) -> None:
    """Delete container and clean up state.

    Workspace is PRESERVED by default (safety measure).

    Args:
        case_number: Case number
        remove_workspace: If True, also delete workspace directory (DANGEROUS, default: False)

    Raises:
        RuntimeError: If container not found or deletion fails
    """
```

**Implementation:**
1. Get container metadata from state
2. If not in state, raise RuntimeError(f"No container found for case {case_number}")
3. Get container from Podman: container = self.podman.client.containers.get(metadata.container_id)
4. Stop container if running: if container.status not in ("stopped", "exited"), container.stop(timeout=10)
5. Remove container: container.remove() (Podman API)
6. Delete from state: self.state.delete_container(case_number)
7. If remove_workspace=True, delete workspace directory:
   ```python
   import shutil
   if os.path.exists(metadata.workspace_path):
       shutil.rmtree(metadata.workspace_path)
   ```
   Print warning: "WARNING: Deleted workspace at {workspace_path}"
8. Print confirmation: f"Deleted container for case {case_number}"

Handle exceptions:
- Catch podman.errors.NotFound if container already removed externally (still clean up state)
- Catch podman.errors.APIError and wrap in RuntimeError
- For workspace deletion, catch OSError and log warning (don't fail delete operation)

Follow RESEARCH.md code examples for container.stop() and container.remove() syntax.

**status() method:**

```python
def status(self, case_number: str) -> dict[str, Any]:
    """Get container status and metadata.

    Args:
        case_number: Case number

    Returns:
        Dictionary with keys:
        - status: str (running, stopped, exited, missing)
        - container_id: str | None
        - workspace_path: str | None
        - created_at: int | None (Unix timestamp)

    Raises:
        RuntimeError: If Podman query fails
    """
```

**Implementation:**
1. Get container metadata from state: metadata = self.state.get_container(case_number)
2. If not in state, return {"status": "missing", "container_id": None, "workspace_path": None, "created_at": None}
3. Try to get container from Podman: container = self.podman.client.containers.get(metadata.container_id)
4. If found, return:
   ```python
   {
       "status": container.status,
       "container_id": container.short_id,
       "workspace_path": metadata.workspace_path,
       "created_at": metadata.created_at
   }
   ```
5. If podman.errors.NotFound, reconcile state and return missing status

**logs() method:**
```python
def logs(self, case_number: str, tail: int = 50, follow: bool = False) -> str:
    """Get container logs.

    Args:
        case_number: Case number
        tail: Number of lines to show (default: 50)
        follow: If True, stream logs continuously (default: False)

    Returns:
        Log output as string

    Raises:
        RuntimeError: If container not found or logs retrieval fails
    """
```

**Implementation:**
1. Get container metadata from state
2. If not in state, raise RuntimeError(f"No container found for case {case_number}")
3. Get container from Podman
4. Call container.logs():
   ```python
   logs = container.logs(
       stdout=True,
       stderr=True,
       timestamps=True,
       tail=tail,
       follow=follow
   )
   ```
5. Decode bytes to string: return logs.decode('utf-8') if isinstance(logs, bytes) else logs

Follow RESEARCH.md code examples for container.logs() syntax.
