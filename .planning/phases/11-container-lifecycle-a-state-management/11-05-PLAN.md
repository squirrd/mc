**exec() method:**
```python
def exec(self, case_number: str, command: str | list[str], workdir: str = "/case") -> tuple[int, str]:
    """Execute command inside container, auto-restarting if stopped.

    Args:
        case_number: Case number
        command: Command to execute (string or list of args)
        workdir: Working directory for command (default: /case)

    Returns:
        Tuple of (exit_code, output)
        - exit_code: int (0 for success, non-zero for error)
        - output: str (combined stdout/stderr)

    Raises:
        RuntimeError: If container not found or exec fails
    """
```

**Implementation:**
1. Get container via _get_or_restart() helper (see below)
2. Execute command:
   ```python
   exit_code, output = container.exec_run(
       cmd=command,
       stdout=True,
       stderr=True,
       stdin=False,
       tty=False,
       workdir=workdir
   )
   ```
3. Decode output: output_str = output.decode('utf-8') if isinstance(output, bytes) else output
4. Return (exit_code, output_str)

**_get_or_restart() helper (Pattern 4 from RESEARCH.md):**
```python
def _get_or_restart(self, case_number: str) -> Container:
    """Get container, restarting if stopped.

    Args:
        case_number: Case number

    Returns:
        Running container instance

    Raises:
        RuntimeError: If container not found or restart fails
    """
    # Get metadata from state
    metadata = self.state.get_container(case_number)
    if not metadata:
        raise RuntimeError(f"No container found for case {case_number}")

    # Get container from Podman
    container = self.podman.client.containers.get(metadata.container_id)

    # Auto-restart if stopped
    if container.status in ("stopped", "exited"):
        print(f"Restarting container for case {case_number}...")
        container.start()

    return container
```

This helper is reusable for future operations that need auto-restart behavior (attach, logs streaming, etc.).

Handle exceptions:
- Catch podman.errors.NotFound and raise RuntimeError("Container not found")
- Catch podman.errors.APIError and wrap in RuntimeError
- Command execution errors reflected in exit_code, not exceptions

import os

def _get_manager() -> ContainerManager:
    """Get ContainerManager instance (helper to avoid duplication)."""
    podman_client = PodmanClient()
    data_dir = user_data_dir("mc", "redhat")
    os.makedirs(data_dir, exist_ok=True)

    return ContainerManager(podman_client, state_db)
