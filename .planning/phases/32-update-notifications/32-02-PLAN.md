Create `tests/unit/test_banner.py` with comprehensive unit tests. All tests mock network calls and ConfigManager — no real HTTP, no real filesystem.

Follow project conventions: `from unittest.mock import MagicMock, patch`, `pytest-mock` style, descriptive names `test_<function>_<scenario>_<expected>`.

**Import block:**
```python
from unittest.mock import MagicMock, patch, PropertyMock
import pytest
from mc.banner import (
    _already_shown_today,
    _fetch_with_timeout,
    _is_version_invocation,
    _render_banner,
    _write_suppression_timestamp,
    show_update_banner,
)
```

**Class `TestIsVersionInvocation`:**
- `test_returns_true_when_version_flag_first_arg`: `sys.argv = ['mc', '--version']` → True
- `test_returns_false_when_subcommand_before_version`: `sys.argv = ['mc', 'case', '--version']` → False
- `test_returns_false_when_no_args`: `sys.argv = ['mc']` → False

**Class `TestAlreadyShownToday`:**
- `test_returns_false_when_none`: ConfigManager mock returns `last_banner_shown=None` → False
- `test_returns_true_when_today`: stored value is today's ISO date string → True
- `test_returns_false_when_yesterday`: stored value is yesterday's ISO date string → False
- All three patch `mc.config.manager.ConfigManager` at source

**Class `TestFetchWithTimeout`:**
- `test_returns_version_on_success`: patch `mc.banner._fetch_latest_version` to return `"2.0.5"` → returns `"2.0.5"`
- `test_returns_none_on_timeout`: patch `mc.banner._fetch_latest_version` with `time.sleep(5)` side effect (or use a threading.Event that never fires) and set `_TIMEOUT_SECONDS` to 0.01 → returns None, stderr contains "timed out"
- `test_returns_none_on_network_failure`: patch `_fetch_latest_version` to return `None` → returns None

For the timeout test, monkeypatch `mc.banner._TIMEOUT_SECONDS` to 0.01 to avoid slowing the test suite.

**Class `TestWriteSuppressionTimestamp`:**
- `test_writes_timestamp_when_tty`: patch `sys.stdout.isatty` → True; verify `ConfigManager().update_version_config` called with `last_banner_shown` kwarg matching ISO format
- `test_skips_when_not_tty`: patch `sys.stdout.isatty` → False; verify ConfigManager NOT called

**Class `TestRenderBanner`:**
- `test_renders_standard_banner`: pass `current="2.0.4"`, `latest="2.0.5"`, `pinned=None`; capture stderr; assert "MC Update Available" in output, "2.0.4" in output, "mc-update upgrade" in output
- `test_renders_pinned_banner`: pass `current="2.0.4"`, `latest="2.0.5"`, `pinned="2.0.4"`; assert "pinned at" in output, "mc-update unpin" in output
- Use `capsys` or redirect Console to StringIO: `from io import StringIO; buf = StringIO(); Console(file=buf, ...)`
  - For Rich Console writing to stderr, monkeypatch: `patch("mc.banner.Console")` and inspect call args, OR redirect stderr with `capsys`

**Class `TestShowUpdateBanner`:**
- `test_skips_on_version_flag`: patch `_is_version_invocation` → True; assert `_fetch_with_timeout` never called
- `test_skips_when_not_tty`: patch `sys.stdout.isatty` → False; assert `_already_shown_today` never called
- `test_skips_when_shown_today`: patch `_already_shown_today` → True; assert `_fetch_with_timeout` never called
- `test_skips_when_no_newer_version`: patch `_fetch_with_timeout` → `"2.0.4"`, `get_version` → `"2.0.4"`; assert `_render_banner` never called
- `test_shows_banner_when_newer_available`: patch all dependencies; `_fetch_with_timeout` → `"2.0.5"`, `get_version` → `"2.0.4"`, `isatty` → True, `_is_version_invocation` → False, `_already_shown_today` → False; assert `_render_banner` called once with correct args
- `test_writes_suppression_after_banner`: same setup as above; assert `_write_suppression_timestamp` called once
- `test_skips_when_fetch_returns_none`: `_fetch_with_timeout` → None; assert `_render_banner` never called

Patch `mc.banner._is_version_invocation`, `mc.banner._already_shown_today`, `mc.banner._fetch_with_timeout`, `mc.banner._render_banner`, `mc.banner._write_suppression_timestamp`, `mc.banner.get_version`, and `mc.config.manager.ConfigManager` at their source modules.
