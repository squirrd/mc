```python
"""Update notification banner for MC CLI startup."""
from __future__ import annotations

import sys
import threading
from datetime import date, datetime
from typing import Optional

_TIMEOUT_SECONDS = 1.5
_BANNER_TITLE = "MC Update Available"
_BORDER_STYLE = "yellow"
```

**Public API — single function:**
```python
def show_update_banner() -> None:
    """Show update banner on stderr if a newer version is available.

    - Skips entirely if not a TTY (piped run)
    - Skips if today's date matches last_banner_shown date in config
    - Fetches latest version from GitHub with a 1.5s timeout
    - On timeout: prints brief note to stderr, does NOT write suppression timestamp
    - On network failure: shows styled warning, writes suppression timestamp
    - Skips if mc --version invocation (caller uses _is_version_invocation())
    """
```

**Helper: `_is_version_invocation() -> bool`**
Return True if `--version` is in `sys.argv` and no subcommand precedes it.
Implementation: `return len(sys.argv) > 1 and sys.argv[1] == '--version'`

**Helper: `_already_shown_today() -> bool`**
- Lazy-import `ConfigManager` from `mc.config.manager`
- Call `get_version_config()`, read `last_banner_shown`
- If None → False
- Parse date portion: `datetime.fromisoformat(last_banner_shown).date()`
- Return `parsed_date == date.today()`

**Helper: `_fetch_with_timeout() -> str | None`**
- Uses `threading.Event` + daemon thread to run `_fetch_latest_version()` from `mc.update`
- Thread writes result into a mutable container (`result = [None]`)
- Main thread waits `_TIMEOUT_SECONDS` on the event
- If event is not set: print `"Update check timed out."` to stderr, return None
- If set: return the fetched version string (or None on network failure)

**Helper: `_write_suppression_timestamp() -> None`**
- Lazy-import ConfigManager
- Call `update_version_config(last_banner_shown=datetime.now().isoformat(timespec='seconds'))`
- Only call this when `sys.stdout.isatty()` is True (interactive check — already guarded at `show_update_banner` level, but guard here too for safety)

**Helper: `_render_banner(current: str, latest: str, pinned: str | None) -> None`**
- `pinned` is the active pin version string if pinned, else None
- Use `rich.console.Console(stderr=True)` for all output
- Print a blank line before and after the panel via `console.print("")`
- If `pinned` is None (not pinned):
  ```
  Panel content:
    v{current} → v{latest}
    Run: mc-update upgrade
  ```
- If `pinned` is not None:
  ```
  Panel content:
    v{latest} available (pinned at v{current})
    Run: mc-update unpin to upgrade
  ```
- Panel: `Panel(content_str, title=_BANNER_TITLE, border_style=_BORDER_STYLE)`
- Panel width: `min=50` via `console.print(panel, width=60)` — let Rich handle terminal width naturally (no fixed width override needed; just pass panel to console.print)

**`show_update_banner()` full flow:**
```
1. if _is_version_invocation(): return
2. if not sys.stdout.isatty(): return   # piped run
3. if _already_shown_today(): return
4. lazy-import get_version from mc.version; current = get_version()
5. lazy-import ConfigManager; version_config = ConfigManager().get_version_config()
6. pinned_mc = version_config['pinned_mc']  # 'latest' or version string
7. latest = _fetch_with_timeout()
8. if latest is None: return             # timeout already printed note
   # Note: network failure returns None from _fetch_latest_version — treat as timeout
9. lazy-import Version, InvalidVersion from packaging.version
10. try: is_newer = Version(latest) > Version(current)
    except InvalidVersion: return
11. if not is_newer: return
12. pinned_arg = pinned_mc if pinned_mc != 'latest' else None
13. _render_banner(current, latest, pinned_arg)
14. _write_suppression_timestamp()
```

All lazy imports (ConfigManager, get_version, packaging) are inside function bodies — `banner.py` must import cleanly at module level with no side effects.
