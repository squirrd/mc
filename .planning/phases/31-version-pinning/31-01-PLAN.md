**Add helper `_fetch_latest_version() -> str | None` after `_print_recovery_instructions()`:**
```
Fetches the latest release version from GitHub. Returns the version string (e.g. "2.0.5") or None on any error.
- URL: https://api.github.com/repos/squirrd/mc/releases/latest
- Headers: {'Accept': 'application/vnd.github+json', 'X-GitHub-Api-Version': '2022-11-28'}
- Timeout: 10 seconds
- On success (200): return response.json()['tag_name'].lstrip('v')
- On any exception or non-200 status: return None
- No retry logic — this is a user-facing interactive command, not background polling
```

**Add helper `_validate_version_exists(version: str) -> bool` after `_fetch_latest_version()`:**
```
Checks if a specific version tag exists in GitHub releases.
- URL: https://api.github.com/repos/squirrd/mc/releases/tags/v{version}
- Headers: same as above
- Timeout: 10 seconds
- On 200: return True
- On 404: return False
- On any other status or exception: raise requests.RequestException (caller handles)
```

**Add `pin(version: str) -> ExitCode`:**
```
Agent mode guard (same as upgrade): import is_agent_mode lazily; if agent, print to stderr:
    "mc-update pin/unpin/check is not available in agent mode. To control the mc version
     inside a case container, pin the container image instead."
and return 1.

Strip leading 'v' from version argument before processing: version = version.lstrip('v')

Validate version format: must match r'^\d+\.\d+\.\d+$' (basic semver). If invalid, print
to stderr: "Invalid version format '{version}'. Expected format: 2.0.3 or v2.0.3" and return 1.

Validate version exists on GitHub:
- Call _validate_version_exists(version) in a try/except requests.RequestException block.
  - On RequestException: print to stderr "Cannot validate version: network unreachable.
    Try again when online." and return 1.
- If returns False: print to stderr "Version {version} not found on GitHub releases." and return 1.

If validation passes, import ConfigManager lazily and call:
    ConfigManager().update_version_config(pinned_mc=version)

Print to stdout: "Pinned to {version}. Run mc-update unpin to remove."
Return 0.
```

**Add `check() -> ExitCode`:**
```
Agent mode guard: same message and return 1 as in pin().

Import get_version lazily: from mc.version import get_version
installed = get_version()

Import ConfigManager lazily.
version_config = ConfigManager().get_version_config()
pin_value = version_config['pinned_mc']  # 'latest' means no pin

latest = _fetch_latest_version()  # May return None if GitHub unreachable

Build output lines:
  Line 1: "Version status:"
  Line 2: f"  Installed : {installed}"
  If latest is not None:
    Line 3: f"  Latest    : {latest}"
  Else:
    Line 3:  "  Latest    : unavailable (network error)"
  If pin_value == 'latest':
    Line 4: "  Pin       : none"
  Else:
    Line 4: f"  Pin       : {pin_value}"
  If latest is not None:
    from packaging.version import Version, InvalidVersion
    Try to compare:
      if pin_value != 'latest':
        Line 5: "  Update    : pinned (run mc-update unpin to upgrade)"
      elif Version(latest) > Version(installed):
        Line 5: "  Update    : available"
      else:
        Line 5: "  Update    : up to date"
    Except InvalidVersion:
      skip Line 5

Print all lines to stdout with print().
Return 0.

Note: 'Update' line is omitted entirely when GitHub is unreachable (latest is None).
```
