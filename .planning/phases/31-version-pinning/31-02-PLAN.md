- `test_pin_version_not_found_on_github`: `monkeypatch.delenv("MC_RUNTIME_MODE", raising=False)`. Patch `mc.update._validate_version_exists` to return False. Call `pin("9.9.9")`. Assert returns 1. Assert stderr contains `"not found on GitHub"`.

- `test_pin_github_unreachable`: `monkeypatch.delenv("MC_RUNTIME_MODE", raising=False)`. Patch `mc.update._validate_version_exists` to raise `requests.RequestException("timeout")`. Call `pin("2.0.3")`. Assert returns 1. Assert stderr contains `"network unreachable"`.

- `test_pin_success_writes_config`: `monkeypatch.delenv("MC_RUNTIME_MODE", raising=False)`. Patch `mc.update._validate_version_exists` to return True. Assert `update_version_config` called with `pinned_mc="2.0.3"`. Assert stdout contains `"Pinned to 2.0.3"` and `"mc-update unpin"`.

**class TestUnpin:**

- `test_unpin_no_active_pin`: `monkeypatch.delenv("MC_RUNTIME_MODE", raising=False)`. Patch `mc.update.ConfigManager` so `get_version_config()` returns `{'pinned_mc': 'latest', 'last_check': None}`. - `test_unpin_removes_active_pin`: `monkeypatch.delenv("MC_RUNTIME_MODE", raising=False)`. Patch `mc.update.ConfigManager` so `get_version_config()` returns `{'pinned_mc': '2.0.3', 'last_check': None}` and `update_version_config` is a MagicMock. Call `unpin()`. Assert returns 0. Assert `update_version_config` called with `pinned_mc='latest'`. Assert stdout contains `"Pin removed."`.

**class TestCheck:**

- `test_check_blocked_in_agent_mode`: `monkeypatch.setenv("MC_RUNTIME_MODE", "agent")`. Call `check()`. Assert returns 1. Assert stderr contains `"agent mode"` or `"container image"`.

- `test_check_all_fields_no_pin`: `monkeypatch.delenv("MC_RUNTIME_MODE", raising=False)`. Patch `mc.update._fetch_latest_version` to return `"2.0.5"`. Patch `mc.update.get_version` — no, `get_version` is imported lazily inside `check()`; patch it via `patch("mc.version.get_version", return_value="2.0.4")` or use the lazy import path `patch("mc.update.check")` — actually, since get_version is imported lazily inside check(), patch it at its source: `patch("mc.version.get_version", return_value="2.0.4")`. Call `check()`. Assert returns 0. Assert stdout contains `"Installed : 2.0.4"`, `"Latest    : 2.0.5"`, `"Pin       : none"`, `"Update    : available"`.

- `test_check_up_to_date`: Same setup with `_fetch_latest_version` returning `"2.0.4"` and installed also `"2.0.4"`, no pin. Assert stdout contains `"Update    : up to date"`.

- `test_check_github_unreachable`: Patch `mc.update._fetch_latest_version` to return `None`. Patch `mc.version.get_version` to return `"2.0.4"`. Patch `mc.update.ConfigManager` for no pin. Call `check()`. Assert returns 0. Assert stdout contains `"Latest    : unavailable (network error)"`. Assert stdout does NOT contain `"Update"` line (omitted when latest unavailable).
