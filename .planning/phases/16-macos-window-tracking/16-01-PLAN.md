Add focus_window_by_id() method to MacOSLauncher class following research pattern from 16-RESEARCH.md (Code Examples section, lines 392-457).

Implementation requirements:
- Method signature: `def focus_window_by_id(self, window_id: str) -> bool:`
- Accept window_id as string (cross-platform compatibility with Linux)
- Return True if window found and focused, False otherwise
- Check osascript availability with `shutil.which("osascript")`, return False if missing
- Escape window_id using existing `_escape_applescript()` method (injection prevention)
- Support both iTerm2 and Terminal.app via `self.terminal` conditional

AppleScript implementation for iTerm2:
```applescript
tell application "iTerm"
    activate
    repeat with theWindow in windows
        if (id of theWindow as text) is "{escaped_id}" then
            -- Un-minimize if needed
            if miniaturized of theWindow then
                set miniaturized of theWindow to false
            end if
            -- Bring to front
            set index of theWindow to 1
            return true
        end if
    end repeat
    return false
end tell
```

AppleScript implementation for Terminal.app:
```applescript
tell application "Terminal"
    activate
    repeat with theWindow in windows
        if (id of theWindow as text) is "{escaped_id}" then
            -- Un-minimize if needed
            if miniaturized of theWindow then
                set miniaturized of theWindow to false
            end if
            -- Bring to front
            set index of theWindow to 1
            return true
        end if
    end repeat
    return false
end tell
```

Error handling:
- Wrap subprocess.run() in try/except
- Set timeout=5 seconds (matches existing pattern in _window_exists_by_id)
- Catch subprocess.TimeoutExpired and return False
- Catch generic Exception and return False
- Return bool based on result.stdout.strip() == "true"

Add method after existing _window_exists_by_id() method (around line 308) to keep window ID operations grouped together.

Reference existing patterns:
- _window_exists_by_id() (lines 250-307) for AppleScript structure and error handling
- focus_window_by_title() (lines 138-209) for focus operation patterns

Why this approach: Research confirms `activate` + `set index to 1` is the standard pattern for cross-Space window focusing on macOS. Explicit miniaturized check handles Dock-minimized windows. No accessibility permissions required (unlike AXRaise).
