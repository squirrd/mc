- source = ["src"]

  - "def __repr__"
  - "raise AssertionError"
  - "raise NotImplementedError"

<task type="auto">
  <name>Task 2: Create hierarchical fixture infrastructure</name>
  <files>tests/conftest.py, tests/unit/conftest.py, tests/integration/conftest.py</files>
  <action>
Create three-level fixture hierarchy following patterns from RESEARCH.md:

**Create tests/conftest.py (root-level fixtures):**
Session-scoped fixtures shared across all tests:
- sample_case_number() - Returns "12345678" (used in test scripts)
- mock_api_base_url() - Returns "https://api.access.redhat.com/support/v1"
- mock_sso_url() - Returns SSO token endpoint URL
- make_case_response() - Factory fixture for generating case API responses with variations (case_number, summary, status parameters)

Follow "Pattern 1: Hierarchical Fixture Organization" and "Pattern 4: Fixture Factories" from RESEARCH.md.

**Create tests/unit/conftest.py (unit test fixtures):**
Function-scoped fixtures for unit tests:
- mock_case_data() - Fresh case data dict per test (case_number, summary, account_number, status)
- mock_access_token() - Returns "fake_access_token" string
- api_client() - Returns RedHatAPIClient instance with fake token (imports from mc.integrations.redhat_api)
- mock_env_vars(monkeypatch) - Sets RH_API_OFFLINE_TOKEN environment variable for auth tests

Follow "Pattern 2: HTTP Mocking with responses" preparation patterns from RESEARCH.md.

**Create tests/integration/conftest.py (integration test fixtures):**
Function-scoped fixtures for integration tests:
- workspace_base_dir(tmp_path) - Returns temporary directory for workspace testing
- sample_workspace_params() - Returns dict with base_dir, case_number, account_name, case_summary

Follow "Pattern 3: File System Testing with tmp_path" from RESEARCH.md.

Each fixture should have:
- Descriptive docstring explaining purpose
- Appropriate scope (session for read-only, function for isolation)
- Type hints where applicable (optional, but recommended)

Import statements needed:
- tests/conftest.py: import pytest, import responses
- tests/unit/conftest.py: import pytest, from mc.integrations.redhat_api import RedHatAPIClient
- tests/integration/conftest.py: import pytest
